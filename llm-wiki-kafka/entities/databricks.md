# Entity: Databricks

A fully managed [[apache-spark|Spark]] platform — where the [[kafka-spark-assignment]] processing runs.

- **Community Edition** is free (community.cloud.databricks.com) — the course's recommended way to get a Spark cluster ([[syllabus]]).
- Hosts notebooks (.dbc / .ipynb), provides a managed Spark cluster, and integrates [[mlflow|MLflow]] (set tracking URI to `databricks`) and Databricks AutoML ([[automl-and-tuning]]).
- Native home for [[delta-lake|Delta Lake]].

Appears in: [[batch-stream-processing]], [[apache-spark]], [[syllabus]], [[kafka-spark-assignment]]
