# Entity: Confluent Cloud

Managed [[apache-kafka|Kafka]] as a service — the platform used to host the topic in the [[kafka-spark-assignment]].

- Provisioned via the `ccloud` CLI: select environment → cluster → create API key/secret → create topic → produce/consume ([[batch-stream-processing]]).
- Includes a hosted schema registry for [[apache-avro|Avro]] schemas.
- Per [[syllabus]]: needs a credit card but gives ~$50/mo free credit for 3 months (enough for the course); a local Confluent install is the free alternative.

Appears in: [[batch-stream-processing]], [[syllabus]], [[kafka-spark-assignment]]
