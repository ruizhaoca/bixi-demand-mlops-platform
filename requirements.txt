# IMPORTANT — why these are NOT capped below the latest generation:
# Streamlit Community Cloud currently provisions **Python 3.14**, which only ships
# binary wheels for the latest scientific-stack generation (numpy 2.4, pandas 3.0,

# generation (numpy<2, pandas<2.3, pyarrow<18, scikit-learn<1.6). On Python 3.14

# the deploy failed ("Failed to download and build pyarrow==17.0.0" / pandas built
# from sdist) -> the app showed "Oh no. Error running app."
#
# These exact versions are the set Cloud resolves on Python 3.14; they also ship
# wheels for Python 3.12 (CI + the EC2 Docker image) and are verified end-to-end to
# load the committed model artifacts and run every page of the app.
